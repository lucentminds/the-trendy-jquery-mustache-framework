# the-trendy-jquery-mustache-framework
Less Angular, React, and Vue with more legacy.

## Install

```shell
npm link
```

## Create new project

```shell
ttjmf make project <PROJECT_NAME>
```

## Start project

```shell
docker-compose up -d
curl localhost
```



## Create new widget

```shell
ttjmf make widget <WIDGET_NAME>
```
